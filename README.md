### Menjalankan Streamlit

Setup Environment
```
pip install numpy pandas matplotlib seaborn jupyter streamlit babel
```

Pindah ke direktori dashboard
```
cd dashboard
```

Jalankan Streamlit
```
streamlit run dashboard.py
```
